# 🌍 Global Health Analytics Dashboard

An interactive **Global Health Analytics Dashboard** built using **R Shiny**, designed to visualize regional health trends through dynamic charts and geospatial mapping.

The dashboard provides an intuitive interface for exploring:

* Regional case statistics
* Vaccination trends
* Interactive geographic visualization
* Comparative analytics across global regions

---

# 📌 Project Overview

This project was developed as an interactive analytical dashboard using:

* **R Programming**
* **Shiny**
* **shinydashboard**
* **Plotly**
* **Leaflet**
* **dplyr**

The system enables users to select different world regions and dynamically update:

* Case analytics
* Vaccination analytics
* Geographic health indicators

The dashboard focuses on transforming structured healthcare data into an easy-to-understand visual decision-support interface.

# ▶️ How to Run the Project

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/global-health-dashboard.git
```

---

## 2. Open in RStudio

Open the project folder in:

* RStudio

---

## 3. Install Required Libraries

```r
install.packages(c(
  "shiny",
  "shinydashboard",
  "plotly",
  "leaflet",
  "dplyr"
))
```

---

## 4. Run the Application

```r
shiny::runApp()
```
