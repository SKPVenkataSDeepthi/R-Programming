# 👥 HR Analytics Dashboard

An interactive **HR Analytics Dashboard** built using **R Shiny** to monitor workforce metrics, employee statistics, salary insights, and retention analytics across multiple departments.

The dashboard provides a streamlined analytical interface for exploring:
- Department-wise employee distribution
- Average salary insights
- Employee retention metrics
- Interactive HR reporting workflows

---

# 📌 Project Overview

This project was developed as an analytical HR reporting dashboard using:

- **R Programming**
- **Shiny**
- **shinydashboard**
- **dplyr**

The system enables users to dynamically select departments and instantly visualize key HR metrics through interactive KPI cards and responsive dashboard components.

The dashboard focuses on simplifying workforce analytics and transforming HR data into actionable business intelligence.

---

# 🚀 Features

## ✅ Interactive Department Filtering

Users can dynamically switch between departments including:
- Data
- IT
- HR
- Finance
- Marketing

using a dropdown selector in the dashboard sidebar.

---

## 📊 Workforce Analytics KPIs

The dashboard includes real-time analytical value boxes displaying:

### Employee Count
Tracks the total number of employees within the selected department.

### Average Salary Insights
Displays average salary distribution for workforce compensation analysis.

### Employee Retention Metrics
Visualizes employee retention percentages to support HR performance monitoring and workforce stability analysis.


```r
  "shiny",

  "dplyr"
```
